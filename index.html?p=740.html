    Alt Downloads

    提供 Coding 和 Aliyun OSS 两种备选下载方式供用户使用。相比百度网盘下载，这些下载方式操作起来会略微繁琐，但下载速度会有明显提升。我们不保证 Coding 和 Aliyun OSS 处的镜像文件能够实时更新。

    从 Coding 获取 Dasyatis 的 Arch Linux 安装镜像

    当前可用版本 / Build 262

    第一步：下载下面的五个 Div 文件。若发现链接失效，请在本页下方留言。

    Div. 0
    Div. 1
    Div. 2
    Div. 3
    Div. 4

    第二步：进入 Div 文件所在目录，合并所有 Div 文件并检查镜像文件完整性：

    dst="dasyatis-build262-x86_64.iso"
hash="a577657ffcdfbe8c416ad989850774b4"
cat ${dst}.div* > ${dst}
echo "${hash} ${dst}" > ${dst}.md5sum
md5sum -c ${dst}.md5sum
rm ${dst}.div* ${dst}.md5sum

    从 Aliyun OSS 获取 Dasyatis 的 Arch Linux 安装镜像

    当前可用版本 / Build 262

    使用此方式产生的少量流量费用（0.25 - 0.50 元 / GB）由用户承担。

    点此查看阿里云云产品定价明细（公共云 - 中国大陆区域 - 按量付费 - 外网流出流量）

    第一步：确保你有一个余额充足且有 访问密钥 的阿里云帐号。获取并初始化 Osstuil，设置地域结点为 oss-cn-shenzhen.aliyuncs.com。

    第二步：进入 Ossutil 所在目录，使用 请求者付费模式 获取镜像文件，以 Osstuil 64 为例：

    dst="dasyatis-build262-x86_64.iso"
./ossutil64 cp oss://bobby285271/${dst} ./ --payer=requester
