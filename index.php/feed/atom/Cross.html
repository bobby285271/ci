]]> <![CDATA[Bobby Rong]]> https://static.bobby285271.top/index.php/Cross.html#comment-17 2020-05-14T04:19:43+00:00 2020-05-14T04:19:43+00:00 Bobby Rong https://static.bobby285271.top GDM 使用的字体和 GNOME 使用的字体不一样就很烦躁，发现一个简单的解决方法。
其实就是登录 gdm 用户然后用 gsettings 改字体，毕竟人家似乎也不管你的 /etc/fonts/local.conf。
machinectl shell gdm@ /bin/bash
gsettings set org.gnome.desktop.interface font-name "Noto Sans CJK SC 11"
gsettings set org.gnome.desktop.interface document-font-name "Noto Sans CJK SC 11"
gsettings set org.gnome.desktop.interface monospace-font-name "Noto Sans Mono 11"
