    Manjaro Guide

    发布于 2018-01-01  51 次阅读 本文共866个字

    2020 更新：Arch Linux 赛高！

    Manjaro Linux 是快速的、用户友好的、面向桌面的、基于 Arch Linux 的操作系统。它的一些显著特性包括：一份直观的安装程序、自动硬件检测、稳定的滚动式发布模式、对安装多个内核的支持、用于管理图形卡的特别 Bash 脚本、高度的桌面可配置性。Manjaro Linux 提供 Xfce、KDE、GNOME 桌面作为核心选项，并为高级用户提供一份最小主义的 Architect 版本。用户还可以获得社区支持的多个版本。Manjaro 的社区论坛可提供帮助并充满活力，用户受益其中。

    系统安装

    常见的安装方式包括使用 Calamares 进行安装和使用 Manjaro Architect 进行安装，点击相应的链接查看安装详细教程。
