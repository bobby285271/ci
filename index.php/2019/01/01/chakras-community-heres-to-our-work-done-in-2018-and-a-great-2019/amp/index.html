2020 年更新：Chakra's Community 官方译文是我写的，当时的 Chakra 社区确实活跃。现在也不是说不活跃了，只是有点各忙各的感觉。

我们的团队衷心地祝愿各位朋友新年快乐！

再一次，我们趁着这个机会感谢那些向我们 报告问题、排查漏洞、提供反馈意见、翻译我们的软件、在社区 #help 板块、IRC 频道 和 Matrix 房间 协助他人、捐款 与将我们的发行版散播出去的人们。

回顾过去

二月份的时候，FranzMari、Ram-Z 和 tetris4 在比利时举行的 FOSDEM 2018 发表闪电演讲，仅仅数天之后，我们更新了 GCC 并针对 GCC ABI 的变化重建了大批软件包，这些包 很快就进入了 [testing] 仓库，当这些包被认为是足够稳定的之后，便 推送给了所有用户。

四月份的时候，带有钩子和文件数据库操作 等功能 的 Pacman 5 进入 Chakra 仓库。

七月份的时候，我们接收到了来自 GitLab Inc. 的赞助，他们为我们提供了全功能企业版本 Gitlab 的授权。

八月份的时候，s8321414 和 tetris4 参加了今年在奥地利举行的 KDE Akademy。

十二月份的时候，附带 CK 补丁的 Linux 4.19 进入 [testing] 仓库。同时，八年以来一直用于 Chakra 交流讨论的 Freenode 群组终于得以被注册，team 群组成员可以获得个性化的称呼 @chakra/$nick 以显示其身份，两个机器人也获得了 @chakra/bot/$nick 的称呼。

这一年中，#tutorials 和 #help:faq 板块新增了不少的文档，例如这些：

  • 如何使用 Pacman 搜索、安装、升级、移除软件包
  • 如何使用 Akabei（预览版本）搜索、安装、升级、移除软件包
  • 如何给软件源列表按照速度排序
  • 如何安装 rEFInd
  • 如何启动一个 systemd-nspawn 容器以用于救援
  • 如何在 Firefox 启用 OpenGL 合成以获得更流畅的使用体验和更低的 CPU 占用

展望未来

进入 2019，我们将致力于：

  • 举办第二次的 Chakra 线下会议，并同时举行密钥签署派对。
  • 将我们的 ISO 镜像制作程序流水线化。
  • 引入 ISO 镜像、软件仓库以及软件包的签名机制。
  • 将网络安装功能添加到我们的 Calamares 安装程序。
  • 更好地组织非滚动更新的软件包组。
  • 重新设计我们的网页。
  • 为我们的基础设施寻找日志和监控解决方案。
  • 继续完善我们的文档。

我们是乐于在空闲时间为 Chakra 做出贡献的一群人。持续改进一切，让 Chakra 变得更好是我们的目标。虽然我们无法提供任何保证，但我们一定会尽我们所能！我们希望您可以享受 Chakra 并成为我们不可缺的一部分！
