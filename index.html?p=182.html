新闻与公告

/ Build 270 修复了 XFCE 终端模拟器下等宽字体显示的问题。

/ 自 Build 262 版本开始，SDDM 已被 LightDM 取代。由于设置了自动登录，因此没有预装任何的 Greeter。live 用户已被添加进 autologin 用户组中。

/ 制作了一个 GNOME 3.36 镜像，预装了完整的 gnome 和 gnome-extra 软件包组，供希望体验新版 GNOME 的用户使用。该镜像同样提供官方 Archiso 附带的所有软件包，但在其它软件的选择上会和常规镜像有所不同。

/ 即日起，我们不再为每个镜像分别设置（百度网盘）文件共享，而是直接共享完整的镜像存档目录。需要下载镜像直接点击上方的「百度网盘 / 提取码 hu6p」按钮，输入提取码后选择需要的镜像下载即可（注意：archives 目录中的镜像为早期发布镜像，不推荐使用）。

/ 自 Build 255 版本开始，live 已被添加至 sudoers 列表中，这意味着 live 用户可以使用 sudo 命令来获得管理员权限。
